# 每日一文
## 一个Android小程序，自动爬取www.meiriyiwen.com的文章、音频，并进行缓存

1. 总体采用Tablayout+Fragment嵌套实现。
2. 实现了文章和图片三级缓存
3. 使用NDK对文件名进行加密
4. 使用多线程播放音乐
5. 采用Material Design控件
6. 除网络请求采用Jsoup外，所有内容均自己构写，未使用第三方库
7. 大量使用异步加载，解决了卡顿问题
